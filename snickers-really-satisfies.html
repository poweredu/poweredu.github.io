4. how did you know that the values of the variable really satisfy the equation ​


How to know if the values of the variables satisfy the equation?

Variables are the unknown values in an equation. So when we are solving an equation, we are figuring out the value of x or any other variable to make the statement true. However, we may come up with a correct or wrong value when solving. To know if the value that we have satisfies the given equation, we have to do the checking. To check, we have to substitute the value that we get to the variable and solve the equation.

Example:

Let us now have an example to have a clear understanding of it.

Equation: 4x = 100

For this equation, we have to know the value of x or the number that when multiplied by 4, the answer is 100.

Solution: 4x/4 = 100/4

x = 25

The value that we got is 25. Let us now check if it satisfies the equation.

Checking: 4(25) = 100

100 = 100 ✔

Therefore, we can conclude that 25 is the correct value of x, and it satisfies the equation.
