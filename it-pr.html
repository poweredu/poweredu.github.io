13. Refer to the trapezoid MAZE with median PR If MA = 16 and EZ = 24, what is PR?


Since PR is a median of trapezoid MAZE, it divides the trapezoid into two congruent triangles, AMP and ERZ.

Since the triangles are congruent, we know that the length of the median PR is equal to the average of the lengths of the bases MA and EZ. Therefore:

PR = (MA + EZ) / 2

Substituting the given values, we get:

PR = (16 + 24) / 2

PR = 20

Therefore, the length of the median PR is 20.

28. PQ is parallel to RS. PR and QS are perpendicular to PQ and RS. The ratio of the lengths of PR and QS is : .


Answer:

By drawing the figure and labeling the given information, we can see that triangles PQR and QSR are similar triangles since they share the same angles (90 degrees, 90 degrees, and congruent angles from PQ parallel to RS).

Therefore, we can set up the following ratio of corresponding sides:

PR/QS = PQ/RS

Since PQ and RS are parallel lines, their corresponding segments are proportional:

PQ/RS = QR/SR

Combining these two equations, we get:

PR/QS = QR/SR

We can simplify this by using the fact that QR = PQ - PR and SR = QS - PR:

PR/QS = (PQ - PR)/(QS - PR)

Multiplying both sides by (QS - PR), we get:

PR^2 = PQQS - PRQS

Adding PR^2 to both sides, we get:

PR^2 + PRQS = PQQS

Factoring out PR on the left side, we get:

PR(PR + QS) = PQ*QS

Dividing both sides by QS, we get:

PR/QS = PQ/ (PR + QS)

Therefore, the ratio of the lengths of PR and QS is PQ/ (PR + QS).
