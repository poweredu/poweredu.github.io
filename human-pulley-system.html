24. a system where several pulleys can be used with the same rope is called?​


Answer:

A system where several pulleys can be used with the same rope is called a block and tackle system. This system consists of multiple pulleys, or blocks, that are mounted on a fixed or movable frame, and a single rope that is threaded through the pulleys. The rope is looped back and forth through the pulleys to create a mechanical advantage, allowing the user to lift or move heavy objects with less force. The block and tackle system has been used for centuries in various industries, such as construction, shipping, and transportation, as well as in recreational activities such as rock climbing and sailing.
