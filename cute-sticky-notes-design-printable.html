17. Luke had 135 sticky labels Vicente had 990 sticky labels and Haley had 570 sticky labels. After Haley gave an equal number of sticky labels to both boys Vicente had 4 times as many sticky labels as Luke. How many sticky notes did Haley have left.


270 Sticky notes

The total number of sticky notes that were left for Haley would be 270. After giving an equal amount of sticky notes to Vicente and Luke, Luke will have a total of 285 while Vicente will have 1140 which is four times the number of Luke's sticky notes.

Here is the step-by-step calculation of their sticky notes.

The original number of sticky notes for each of them has:

Luke = 135 sticky notes
Vicente = 990 sticky notes
Haley = 570 sticky notes.

After giving away sticky notes from Haley to the boys, the following descriptions should be applied:

Haley = [tex]570 - x[/tex]
Luke = [tex]135 + \frac{x}{2}[/tex]
Vicente = [tex]990 + \frac{x}{2}[/tex]

Given the above scenario, we can create a linear equation in one variable.

Vicente = 4(Luke)
[tex](990 + \frac{x}{2} ) = (4)(135+\frac{x}{2} )[/tex]

Calculate the equation to get the value of x.

[tex](990 + \frac{x}{2} ) = (4)(135+\frac{x}{2} )[/tex]
[tex](1980 + x) = (8)(\frac{135 + x}{2})[/tex]
[tex](1980 + x) = 1080 + \frac{8x}{2}[/tex]

Reduce the fraction to its lowest term then combine liked terms.

[tex](1980 + x) = 1080 + 4x[/tex]
[tex]1980+x-4x=1080[/tex]
[tex]1980-3x=1080[/tex]

Calculate the value of x

[tex]-3x=1080-1980[/tex]
[tex]-3x=-300[/tex]
[tex](\frac{1}{3} )(-3x)=(\frac{1}{3})(-300)[/tex]
[tex]x = 300[/tex]

The total number of sticky notes that Haley gave to Vicente and Luke is 300. This also means that each of the boys received 150 sticky notes.

To get the number of sticky notes that were left for Haley, we have to subtract the value of x from the original number of sticky notes that Haley had.

Haley = [tex]570 - x[/tex]
          = [tex]570 - 300[/tex]
          = [tex]270[/tex] sticky notes.

Therefore, the correct answer is 270 sticky notes.
