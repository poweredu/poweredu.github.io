9. Use the Fundamental Counting Principle to find the total number of possible outcomes. Meal Appetizer, Spinach dip, Salad Entrée Chicken, Beef, Spaghetti, Fish Dessert Yogurt, Fruit, Rice pudding There are total possible outcomes.


Answer:

The Fundamental Counting Principle states that if a task has n number of options for the first choice, m number of options for the second choice, and p number of options for the third choice, the total possible outcomes are n x m x p. In this example, there are 3 choices for the appetizer (n = 3), 4 choices for the entrée (m = 4), and 3 choices for dessert (p = 3). Therefore, the total number of possible outcomes is 3 x 4 x 3 = 36.
