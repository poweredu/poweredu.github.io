Programming

1. How do you apply text color (yellow) and font name (Algerian) on the heading tag <h2>? Give an example.

The code for applying text color is...

Inline Styling: <h2 style:"color: yellow; font-family: algerian;">Hello World</h2>Internal CSS/Styling: <style>h2 {color: yellow; font-family: algerian;}</style>External CSS: h2 {color: yellow; font-family: algerian;}

I attached images on how to type these codes in your programming software or code editor. See Images #1 and #2.

2. How do you apply background color (blue) on the body and it displays black color on the webpage? Give an example.

Follow these codes (Note: Purely HTML; without CSS):

<!DOCTYPE html>

<html lang="en" dir="ltr">

<head>

 <meta charset="utf-8">

 <title>Practice #1</title>

 <style media="screen">

   body {

     background-color: blue;

   }

 </style>

</head>

<body style="background-color: black;">

</body>

</html>

The preview image of the written code and the result is in image #3.

Additional Info:

The color you have specified in your HTML file or CSS file will always be the color you will see on your webpage. Except when you wrote a wrong code that makes the default color overrides your written color code. Or unless you wrote multiple codes that describe the <body> tag's color where the black color overrides the blue one because of the hierarchy of the styling you used.

For example, in terms of the hierarchy of the background color tags...

The first style the browser will accept is the inline style. It is the default style. When you have other color tags for the same element, your browser will follow the inline style and override the other.The second one will be the body style or the internal CSS. Even if you have written a color code for a specific element in your CSS [external] file and you also wrote another color style for that same element in your HTML, the latter will override the former. The last in the hierarchy is the external CSS. This is where a programmer puts all of the styles he/she used for his HTML file. If there are no tags with a higher hierarchy in that specific element, no overriding will happen and the browser will follow the specified style in the CSS file.  

On the other hand, the external CSS overrides the default styles of the browser.
