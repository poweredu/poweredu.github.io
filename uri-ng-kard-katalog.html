6. Paano nagkakatulad ang tatlong uri ng kard sa katalog


Ang katalogo ng aklatan, muwestraryong pang-aklatan, tarhetang katalogo o kard katalog ay isang listahan ng lahat ng nilalaman ng isang aklatan, na nakaayos gamit ang isang tarheta o kard para sa bawat isang bagay na makikita sa aklatan. Noong huli ng ika-20 siglo, ang mga pisikal na katalogo ng aklatan ay napalitan ng mga naka-computerize na bersyon. Mga iilang aklatan na lamang ay gumagamit na pisikal na katalogo ng aklatan: ito ay karaniwan sa mga maliliit at malalayong aklatan.

7. Takdang aralin Sumulat / gumawa ng halimbawa ng dalawang uri Ng kard katalog​


Answer:

1. Alfabetikal na Ito ang pinaka-karaniwan pagsasama ng mga impormasyon tungkol sa isang item o koleksyon. Halimbawa, isang aklatan ay maaaring magkaroon ng alfabetikal na katalogo upang matulungan ang mga gumagamit na hanapin ang tamang titulo ng aklat.

2. Tematikal na katalogo: Ito ay nagbibigay ng isang pamamaraan upang maihagis ang mga item sa mga tema para sa mas madaling pag-access. Halimbawa, isang museo ay maaaring gumamit ng tematikal na katalogo upang mapagkalooban ang mga bisita ng maayos na pag-organisa ng mga eksibit.

Answer:

Uri ng Kard Katalog

Kard ng manunulat/May akda- tinatawag itong pangunahing tala, itoy nakaayos ng paalpabeto mula sa unang titik hanggang huling letra.

Kard ng Paksa- Nakaayos ito rin ito ng paalpabeto ayon sa unang titik ng paksa ng aklat.

19. ano ang kard katalog at uri nito


Ang kard katalog ay naglalaman ng mga listahan ng nilalaman ng isang babasahin o aklat. Ang kard katalog ang mag ibat-ibang uri. Ang kard katalog ay may dalawang uri ng klasipikasyon. Ang kard katalog ay napakahalaga upang mahanap agad ang bahagi ang libro. Eto ay nakaayos gamit ang tarheta o card na may sukat na 3x5.

Uri Ng Kard Katalog

Ang mga sumusunod ay ang mga uri ng kard katalog:

Kard ng manunulat o ng may akda Kard ng Paksa Kard ng pamagat
