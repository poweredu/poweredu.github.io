4. 2. Do you think interactive multimedia has an impact on the development of web based applications? Explain your answer.


INTERACTIVE MULTIMEDIA

Interactive multimedia allows the user to control, combine, and manipul4te a variety of media types, such as text, computer graphics, audio and video materials, as well as animation. Interactive multimedia integrates computer, storage, data, phone, TV, and other information technologies.

Interactive multimedia is a form of multimedia content that allows users to more interactively contribute to the multimedia content. Interactive here is the user can control, comment on and add elements that exist in a Multimedia content using digital manipul4tion.[1]

Interactive multimedia can be useful for teachers and students because it makes the learning process more interesting, more interactive, the quality of student learning increases, and the learning process can be done any’where and anytime.

Another form of Interactive Multimedia is Hypermedia, Hypermedia is an advanced form of Interactive Multimedia. The difference with ordinary Interactive Multimedia is that Hypermedia allows its use to be able to contribute more to the Multimedia content.

If the usual form of Interactive Multimedia, users can only contribute through control, comment and add certain elements. So Hypermedia allows users to be more free' to change and add elements in any content, regardless of the form of the elements.

Advantages:

Can be designed according to individual wishes. Can be used independently.Help increase learning motivation. Can provide feedback or response to users.

Disadvantages:

The need for teamwork to develop.The development process takes quite a long time
