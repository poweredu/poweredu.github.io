Ano Ang Neokolonyalismo?

NEOKOLONYALISMO ay ang impluwensyang panlipunan at pangkultura ng mga mananakop. Ito’y impluwensya na walang ginamit na military o pulitikal na kontrol para makamit.

Bukod dito, ang salitang “neo” ay nangangahulugang “makabago”. Kaya naman, kung ating titignan ang buong salita, ang “neokolonyalismo” ay ang pagsakop ng isang mas makapangyarihang bansa sa isang bansa na mahina at inabuso ito sa ekonomikal na paraan.

Mga pamamaraan at uri ng neokolonyalismoAng pamamaraan ng neokolonyalismo

1. Ito ay ang makabagong uri ng pananakop upang mapanatili ang kanilang kapangyarihan. Pagpapanatili ng kapangyarihan ng isang dating kolonyalista  

Subtle

Malumanay at patagong pamamaraan at pagmamanipula sa isang bansa upang patatagin ang pamumuhunan, pigilin ang paggamit ng tunay na Kalayaan at kunin ang mas malaking kita mula sa negosyo.Palakasin ang imperyalismo sa usaping pang-ekonomiya, pulitika, ideolohiya at military na paninindigan ng bansa.

Pagkakaroon ng makabagong pamamaraan para sa pagluluwas ng pamuhunang industriyal at pinansiyal.

•  Kabilang dito ang pagbuo ng iba’t-ibang uri ng lipunan at kumpanya, pandaigdigan at pampribadong pondo, pagkakaroon ng mga konsorsyum, samahan ng mga namumuhunan , pasisiguro ng mga namumuhunan, paglalaan ng mga mga pautang  na makakatutulong hindi lamang sa mga nangangailangan kundi magbigay din ng sapat na tubo sa pamamagitan ng magpapahiram.

Nakatuon ito sa kita ng kapital na kanilang inilagay sa mga negosyo sa mga bansang papaunlad pa lamang.

URI NG NEOKOLONYALISMO

1. Ekonomiko

Kabilang dito ang paggawa ng iba’t-ibang uri ng lipunan at kumpanya, pandaigdigan at pampribadong pondo, pagkakaroon ng mga konsorsyum, samahan ng mga namumuhunan , pasisiguro ng mga namumuhunan, paglalaan ng mga mga pautang  na makakatutulong hindi lamang sa mga nangangailangan kundi magbigay din ng sapat na tubo sa pamamagitan ng magpapahiram.

2. Kultural

• Ito ang naging daan upang mabago ang isang bansa. Nagkaroon ng pagbabago sa buhay ng tao at higit nilang pinahalagahan ang  musika, palabas at babasahin ng mga dayuhan.

27. ano ano ang mga uri ng NEOKOLONYALISMO


Neokolonyalismo

Ang salitang neokolonyalismo ay ginagamit upang ilarawan ang isang makabagong uri ng kolonyalismo. Dito, kinokontrol ng isang kolonisador na bansa ang pamumuhay ng mga tao sa bansang kolonya. Pinanghihimasukan din ng mga kolonisador ang pamamahala at gobyerno ng kanilang kolonya.  

Mga uri ng neokolonyalismo Ekonomiko - panghihimasok sa pamamalakad ng ekonomiya ng isang bansa Kultural - pagyakap ng isang bansa sa mga banyagang kultura Gobyerno - pinanghihimasukan ang pamamahala sa isang bansa

Mga tanda ng pagkakaroon ng neokolonyalismo

Dahil makabago na ang panahon ngayon, makabago na rin ang uri ng pananakop ng mga bansa. Bagamat ito ay hindi direktang pagsakop katulad ng paggamit ng dahas, ito ay pagpapailalim naman sa kapangayarihan ng kolonisador. Narito ang ilan sa mga tanda ng pagkakaroon ng neokolonyalismo sa isang bansa

Ang bansa ay nagiging tambakan ng mga surplus o sobrang produkto Ang bansa ang nagiging pangunahing pinagkukunan ng lakas paggawa sa murang halaga Ang bansa ay tinatapunan ng mga nakalalason na basura Pagiging tambakan ng isang bansa ng mga nakapipinsalang basura Nagiging taga supply ng mga hilaw na materyales
