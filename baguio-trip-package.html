1. A travel agent is selling 75 discount packages. He makes P 2,500 for each Vigan Tour Package and P 2,800 for Baguio Tour Package. How much will he makes if he sells 27 Vigan Tour Packages and 45 Baguio Tour Package?


Answer: A total of Php 151,425

Step-by-step explanation:

75%(2500)=Php 1875 each VTP

75%(2800)=Php 2100 each BTP

27 VTP × Php 1875=Php 50625

45 BTP × Php 2100=Php 100800
