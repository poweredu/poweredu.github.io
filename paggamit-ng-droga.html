12. Mga dahilan sa paggamit ng droga


Answer:

BAKIT GUMAGAMIT NG DROGA ANG MGA TAO?

May iba't ibang dahilan kung bakit nag ddroga ang mga tao.

Narito ang ilan sa mga dahilan sa paggamit ng mga droga:

1. Para makibagay - madalas ito ay impluwensya ng mga tao sa paligid nila. Pakiramdam nila na dapat nilang gawin din ang pagddroga dahil ito ang uso.

2. Para “makatakas” o makapag-relaks - minsan ay dahil sa problema na kinakaharap ng isang tao, pakiramdam nila ay wala ng magiging solusyon kaya para makalimot o mapakalma ang sarili ay gagamit sila ng droga.

3. Para mapawi ang pagkabagot - ang iba naman ay "trip" lang. Ginagawang libangan.

4. Para magrebelde - ilan sa kabataan na hindi nabibigyan ng atensyon na ninanais nila ay nagddroga. Niiisip nila na sa gantong paraan ay makukuha nila ang atensyon ng tao sa paligid nila.

Iba't iba man ang dahilan, akala nila na ang droga ay solusyon. Hindi nila alam na kalaunan, ang droga ang kanilang magiging problema pisikal at mental.


13. Isyu tungkol sa paggamit ng droga


Maraming isyu tungkol sa paggamit ng droga. Pero ang ilan ay ating tatalakayin.

Una, ang droga dito sa Pilipinas ay pinagmumulan ng exta judicial killing. Isang bagay na talagang nagiging alarma sa human rights hindi lamang sa Pilipinas kundi maging sa United Nations.

Totoo, talagang aasahan ang ganitong mga pagpatay, pero talaga nga kayang handa ang kapulisyahan na sugpuin ito o ginagawang front lamang ang maliliit na mga tao upang maiiwas sa tunay na drug lord? Talaga nga kayang hindi alam ng mga nasa awtoridad ang tunay na salarin nito anupat nagagawa pang magsagupaan at maghulihan ng mga nasa ibaba ng sindikato? Talaga nga kayang ang mga tao na humuhuli ay hindi sangkot sa isyu ng droga? Isang nakababahalang pagbagsak ng katarungan.

Ikalawa, ang paraan ba ng pagsugpo sa adiksyon sa droga ay sumasabay sa kahandaan ng tahanan at mga rehabilitasyon? Ang tahanan ang masasabing preventive steps upang hindi na maipasok sa adiksyon ang kanilang mga anak. Ang problema, humihina ang paggalang as awtoridad ng mga magulang dahil sa kakulangan ng komunikasyon. Ang mga rehabilitation centers naman ang gumagamot dito. Pero ipinakikita na ang ilang centers ay kulang sa kasanayan at pasilidad.

Ikatlo, ang kakulangan ng tunay na layunin ng buhay ang siyang nagpapahina sa moralidad ng tao. At ang adiksyon ay isang palatandaan ng pagkaalipin ng  tao mula sa kasalanan. Pero maraming nagbago ang buhay dahil sa paghahanap ng sagot sa kanilang mga tanong at mga pangangailangan. Kailangan lamang ituro ito sa mga nangunguna at iakay ang mga biktima nito. Ang isyu dito, kamusta ang espiritwalidad ng mga nasa awtoridad?

30. ano ang kahulugan ng paggamit ng droga


Answer:

Ang mga droga ay mga lason. Ang dami na ginamit ang makatitiyak sa resulta.   Ang kaunti ay nagsisilbing pampasigla (nagpapabilis sa iyo). Ang mas marami ay nagsisilbing sedatibo (pinababagal ka). Ang mas marami pa ay nakalalason at maaaring makamatay.

Ito ay totoo sa kahit anumang droga. Tanging ang daming kinakailangan para makamit ang epekto ang naiiba.  Ngunit maraming droga ang may ibang panganib: ang mga ito ay direktang nakaaapekto sa isipan. Maaaring pasamain ng mga ito ang pag-unawa ng gumagamit sa nangyayari sa kanyang paligid. Bilang resulta, ang mga kilos ng taong iyon ay maaaring maging kakaiba, hindi makatwiran, hindi naaangkop at maaari pang mapanira.

Hinaharangan ng mga droga ang lahat ng mga pandama, ang mga kanais-nais ay pinapalitan ng  mga hindi kanais-nais. Kaya, habang nagdudulot ng panandaliang tulong sa pagpapaginhawa ng sakit, tinatanggal din ng mga ito ang kakayahan at kalistuhan at pinalalabo ang pag-iisip.

Ang mga medisina ay mga drogang inilayon para mapabilis, mapabagal o mabago ang isang bagay tungkol sa paano gumagana ang iyong katawan, para subukang mapahusay ang paggana nito. Minsan ay kinakailangan ang mga ito. Ngunit ang mga ito ay mga droga pa rin: kumikilos ang mga ito bilang mga pampasigla o mga sedatibo, at ang sobra ay makamamatay sa iyo. Kaya’t kapag hindi mo gagamitin ang mga medisina tulad ng kung papaano dapat gamitin ang mga ito, maaaring maging kasing-panganib ng ilegal na mga droga ang mga ito.

BAKIT GUMAGAMIT NG DROGA ANG MGA TAO?

Gumagamit ng droga ang mga tao dahil may gusto silang baguhin sa buhay nila.

Heto ang ilan sa mga dahilan sa paggamit ng mga droga na ibinigay ng mga kabataan:

Para makibagay Para “makatakas” o makapag-relaks Para mapawi ang pagkabagot Para maging parang matanda Para magrebelde Para mag-eksperimento Akala nila na ang mga droga ay solusyon. Ngunit kalaunan, ang mga droga ang nagiging problema.
