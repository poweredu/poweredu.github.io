Answer:

Formula y = (n + 2) x n

Let n be  Angelica's age and (n + 2) be Lady Ann's age.

y = 528

528 = (n + 2) x n

528 = n^2 + 2n

528 - n^2 - 2n = 0

- 528 + n^2 + 2n = 0

n^2 + 2n - 528 = 0

n^2 + 24n - 22n - 528 = 0

n(n + 24) -22(n + 24) = 0

(n + 24) x (n - 22) = 0

n + 24 = 0

n - 22 = 0

n = -24 n = 22

Therefore Angelica's age is 22 and Lady Ann is 24.
