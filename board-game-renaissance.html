19. The width of a rectangular board game is 10 inches. Its perimeter is 56 inches. Find the length and area of the board game


Answer:

Length = 18 inches

Area = 180 in²

Step-by-step explanation:

Given,

w = 10 inches

P = 56 inches

Solution for the length:

P = 2 (length + width)

56 = 2 (L + 10)

56 = 2L + 20

2L = 56 - 20

2L = 36

L = 18

The length of the rectangular board game is 18 inches.

Solution for the area:

Area = L(w)

Area = 18(10)

Area = 180 in²

The area of the rectangular board game is 180 in².
