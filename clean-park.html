11. The median number of parks cleaned by 10 service workers in a certain year is 4. The range of the number of parks cleaned by those service workers that year is 7. Determine if the following statement is true, is false, or does not contain enough information. At least one service worker cleaned 4 parks.


Answer:

Not enough information

Step-by-step explanation:

The range of a set is the highest minus the lowest element, meaning that the highest - lowest number in this set is 7. Although this is so, this does not indicate much on the values between them.

Moreover, even if the median number of parks is 4, it is possible that there is no value 4 in the element but instead 3 and 5. Having 10 values, it is possible that the set would look like the following if arranged in an ascending order:

_ _ _ _ 3 5 _ _ _ _

Both the presence and absence of 4 is possible. Therefore, there is not enough information to prove that one service worker cleaned for parks.
