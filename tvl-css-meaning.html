1. scope and delimitation to tvl ict css​


Answer:

The scope of the TVL ICT CSS (Technical Vocational Livelihood Information and Communication Technology Computer Systems Servicing) program includes the study of hardware and software installation, configuration, and maintenance. The program also covers computer networking, web development, and database management. The students will learn the necessary skills to become computer technicians, network administrators, and IT support staff.

The delimitations of the TVL ICT CSS program include the restriction of the program to the study of computer systems servicing and related fields. It does not cover other areas of information technology such as programming or software development. The program also focuses on the technical skills needed for entry-level positions in the field, rather than on the management or strategic aspects of IT. The program may also have limitations in terms of available resources, equipment, and facilities, which may impact the depth and breadth of the curriculum.

9. what is the meaning of CSS


What is the meaning of CSS?

CSS means Cascading Style Sheets. It is used to make HTML elements displayed properly and orderly in a webpage. Mostly, HTML focuses on the website contents like what could be read on the website's body, or what you could see in the video and image box while CSS focuses on the style of the web page. Such style includes background color, font styles, colors and face and other things.

CSS cannot exist without HTML since it is based on the HTML elements. For example, you wanted a webpage with two paragraphs has different font color and style, instead of creating several tags for them, you can just create one file for these two HTML elements to style them. CSS works that way, to make styling of web pages easier.
