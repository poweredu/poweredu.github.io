16. nine times the square of a number is 28 more than nine times the number, what is the number? With solution please. ​


SOLVING THE ROOTS OF QUADRATIC EQUATION

Problem:

Nine times the square of a number is 28 more than nine times the number. What is the number?

Answer and Step-by-step explanation:

Let x be the number.

Nine times the square of the number is equivalent to [tex]9x^{2}[/tex].

28 more than nine times the number is equivalent to [tex]28[/tex]+[tex]9x[/tex].

The word "is" in the given mathematical sentence means an equal sign "=".

If we will translate this to an algebraic equation, we will have:

[tex]9x^{2} = 28 + 9x[/tex]

With this, we can solve for the value of x by transposing 28 and 9x to the left side of the equation. Remember that their signs will change after we transpose them so 28 will become negative and so with 9x. This will result to:

[tex]9x^{2}-9x-28=0[/tex]

We can use quadratic formula method to find the values of x.

[tex]x = \frac{-b+/-\sqrt{b^{2}-4ac} }{2a}[/tex]

where: a is 9, b is -9 and c is -8.

[tex]x = \frac{-(-9)+/-\sqrt{(-9)^{2}-4(9)(-28) } }{2(9)}[/tex]

[tex]x = \frac{9+/-\sqrt{81-(-1008) } }{18}[/tex]

[tex]x = \frac{9+/-\sqrt{1089 } }{18}\\x = \frac{9+/-(33) }{18}\\\\x_{1} = \frac{9+33}{18} \\\\x_{1} = \frac{42}{18} \\\\x_{1} = \frac{7}{3}[/tex]

Solving for the second value of the root:

[tex]x_{2} = \frac{9-33}{18} \\\\x_{2} = \frac{-24}{18} \\\\x_{2} = \frac{-4}{3}[/tex]

Therefore, there are two values of x which is 7/3 and -4/3.
