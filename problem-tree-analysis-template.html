23. Explain different tools used in the different stages of scope management: 1. Meeting minutes template 2. Questionnaires and surveys 3. Prototypes 4. Variance analysis tools (e.g., spreadsheets)


1. Meeting minutes template: This tool is typically used in the planning stage of scope management. During meetings, notes are taken and compiled into a meeting minutes document. This document serves as a reference to help ensure that everyone is on the same page and understands what was discussed and decided upon.

2. Questionnaires and surveys: This tool is usually used in the initiation stage of scope management. These tools help to gather information from stakeholders, such as their requirements and expectations. This information can then be used to define the project scope and identify potential issues that need to be addressed.

3. Prototypes: Prototyping is a tool that is often used in the design stage of scope management. A prototype is a preliminary model or version of the final product. This tool enables stakeholders to visualize the product design and provide feedback that can be used to refine and improve the final product.

4. Variance analysis tools (e.g., spreadsheets): Variance analysis tools are typically used in the control stage of scope management. These tools help to identify differences between actual project performance and the plan. For example, by using spreadsheets to monitor and analyze project costs, project managers can identify areas where the project is over or under budget and take corrective action.
