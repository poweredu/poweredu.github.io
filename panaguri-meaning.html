1. Meaning of the simuno at panaguri


Answer:

Ang simuno ang panaguri ay ang mga bahagi ng isang pangungusap.

Simuno

Ang simuno ang siyang nagsasabi kung ano o sino ang pinag-uusapan sa pangungusap. Ito ang paksa ng pangungusap.

Halimbawa:

1.Ang mga lobo ay makukulay.

Ano ang makukulay? Ang mga lobo

2.Si Martha ay naghuhugas ng pinggan.

Sino ang naghuhugas ng pinggan? Si Martha

3. Nagtatanim ng palay sina Lukas at Tonyo.

Sinu-sino ang mga nagtatanim ng palay? Sina Lukas at Tonyo

Panaguri

Nagsasabi tungkol sa simuno o paksa ang panaguri.

1.Ang mga lobo ay makukulay.

Ano ang sinasabi tungkol sa mga lobo? makukulay

2.Si Martha ay naghuhugas ng pinggan.

Ano ang sinasabi tungkol kay Martha? naghuhugas ng pinggan.

3. Nagtatanim ng palay sina Lukas at Tonyo.

Ano ang ginagawa nina Lukas at Tonyo? Sina Lukas at Tonyo

10. what does panaguri and simuno mean? ​


Answer:

I. Panaguri

Ito ay bahagi ng pangungusap na kung saan nagsasabi ng tungkol sa simuno.

  Halimbawa:

  Ang titser ay nagtuturo

II. Uri ng Panaguri

Panaguring Pandiwa

Ito ay panaguri ng pangungusap ay pandiwa.

Halimbawa:

Si James ay nagaaral mag maneho.

Siya ay bumili ng pataba.

  2.  Panaguring Pang-uri

Ang panaguring pang-uri ay maaring isang salita o isang parirala.

Halimbawa:

Malinamnam ang manggang hinog.

Nakakatakot ang boses ni Cris.

 3. Panaguring Pangngalan

Ang pangngalang ginagamit sa panaguri ay nagpapakilala kung sino o ano ang simuno.

Uri ng Panaguring Pangngalan

     a. Tiyak

Kapag ang ginagamit na panaguri ay pantangi.

 Halimbawa:

  Tinawag ng doctor si Eula  

   b. Di-Tiyak

Kapag ang panaguri na ginagamit ay pambalana.

Halimbawa:

Umawit si Sarah ng kundinman

  4. . Panaguring Panghalip

Ito ay humahalili sa pangngalan.

Halimbawa:

Ito ay libro ni Roger.

Siya ang puno ng barangay.

  5. Panaguring Pang-abay

Halimbawa:

Kanina pa umalis si Kristina.

Sa Canada nagbakasyon ang pamilya.

  6. Panaguring Pawatas

Halimbawa:

Manggamot ang naging trabaho niya sa kabilang barrio.

III. Simuno

Ito ay ang paksang pinag-uusapan sa pangungusap.

Halimbawa:

Naninirahan ang mga Mangyan sa lalawigan ng Mindoro.

Answer:

Ang panaguri (Ingles: predicate[1]) ay isang bahagi ng pangungusap o pananalita. Ang panaguri ay isa sa dalawang pangunahing bahagi ng pangungusap kasama ang simuno o paksa. Sa wikang Filipino, kapag nasa karaniwang ayos ang pangungusap, nauuna ang panaguri habang nasa huli naman ang panaguri kapag nasa kabalikang ayos.
