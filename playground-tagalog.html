22. The city government is planning to construct a new children’s playground. It wants to fence in a rectangular ground using one of the walls of a building. The length of the new playground is 20 meters longer than the width. The area of the playground is at least 2,925 square meters. a. What would be the minimum width of the playground? b. What would be the minimum length of the playground?


Answer:

Let's first set up an equation to represent the area of the playground. We can call the width of the playground w and the length l. Since the length of the playground is 20 meters longer than the width, we can write the equation as:

Area = w * (w + 20)

Since the area of the playground is at least 2,925 square meters, we can set this equation equal to 2,925 and solve for w:

2,925 = w * (w + 20)

We can use the quadratic formula to solve for w:

w = (-b +- sqrt(b^2 - 4ac)) / 2a

Where:

a = 1

b = 20

c = -2,925

Substituting these values into the formula and solving, we get:

w = (-20 +- sqrt(20^2 - 4 * 1 * (-2,925))) / 2 * 1

w = (-20 +- sqrt(400 + 11,700)) / 2

w = (-20 +- sqrt(12,100)) / 2

w = (-20 +- sqrt(110 * 110)) / 2

w = (-20 +- 110) / 2

w = (-130, 90) / 2

Since the width of the playground must be a positive number, we can disregard the negative solution, and our final answer is:

w = 90 / 2

w = 45

Therefore, the minimum width of the playground is 45 meters.

To find the minimum length of the playground, we can use the equation l = w + 20, where w is the width of the playground that we just found. Substituting our value of w = 45 into this equation, we get:

l = 45 + 20

l = 65

Therefore, the minimum length of the playground is 65 meters.


23. a playground has a length that is 24 meters shorter than thrice it's width.the are of the playground is 720 square meters.what are the dimensions of the playground​


Problems Involving Algebraic Expressions

To solve this problem, we need to represent the unknown using any variable. Form the algebraic expression using the given. Make sure to analyze the problem clearly so that the correct operation will be used. Look for clues example the word thrice which means tripled. Check the answers obtained by using the algebraic expressions formed.

Answer:

length = 36 square meters

width = 20 square meters

Solutions:

Let: x - the length of the playground

then,

3x - 24 - width of the playground

Given: 720 square meters - area of the playground

A = l x w

720 = (x)(3x - 24)

720 = 3x² - 24x

Write the equation in standard form.

3x² - 24x - 720 = 0

Simplify the equation.

x² - 8x - 240 = 0

By factoring,

x² = (x)(x)

-240 = (1)(240), (-1)(240), (2)(120), (-2)(120), (3)(80), (-3)(80), (4)(60), (-4)(60), (5)(48), (-5)(48), (6)(40), (-6)(40), (8)(30), (-8)(30), (10)(24), (-10)(24), (12)(20), (-12)(20), (15)(16), (-15)(16)

Choose the factor of -240 that will give a sum of -8 which is the middle term of the equation x² - 8x - 240 = 0.

-240 = (1)(-240), (-1)(240), (2)(-120), (-2)(120), (3)(-80), (-3)(80), (4)(-60), (-4)(60), (5)(-48), (-5)(48), (6)(-40), (-6)(40), (8)(-30), (-8)(30), (10)(-24), (-10)(24), (12)(-20), (-12)(20), (15)(-16), (-15)(16)

Thus,

x² - 8x - 240 = (x + 12)(x - 20)

(x + 12)((x - 20) = 0

x + 12 = 0; x - 20 = 0

x = -12; x = 20

Choose the one that is positive since dimensions are expressed in positive measurements.

x = 20

width = 20

Therefore, the width of the playground is 20 meters.

Find the length.

length = 3x - 24

length = 3(20) - 24

length = 60 - 24

length = 36

Therefore, the width of the playground is 36 meters.

720 square meters = 36 meters x 20 meters

720 square meters = 720 square meters

29. A rectangular playground is 36 meter wide and 46 meters long. What is the perimeter of the playground?​


Answer:

The perimeter of the rectangular playground is calculated by adding the length of all its sides. In this case, the length of each side is equal to the sum of the width and the length of the playground, which is 36 + 46 = 82 meters. Therefore, the perimeter of the rectangular playground is 2 * 82 = 164 meters.
