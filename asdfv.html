4. 3.find the perimeter of pentagon ASDFG with vertices A(-2,3)S(-2,1)D(1,-1)F(-2,2)G(-4,0)​


In the given problem, apply your knowledge in algebra and basic mathematics specially in solving word problem about distance of points, and perimeter of a figure.

Given

ASDFG with vertices A(-2,3), S(-2,1), D(1,-1), F(-2,2), and G(-4,0)

Problem

Find the perimeter of the pentagon.

Solution

Use the distance formula to identify the lenght of each side, and the formula for solving perimeter of a pentagon.

Solve for Pentagon Sides

d = √(x2-x1)² + (y2-y1)²

Side AS, A(-2,3), S(-2,1)

S1 = √(-2-(-2)²+(1-3)²

S1 = √0+4

S1 = √4

S1 = 2

Side SD, S(-2,1), D(1,-1)

S2 = √(-2-1)²+(-1-1)²

S2 = √(-3)²+(-2)²

S2 = √9+4

S2 = √13

Side DF, D(1,-1), F(-2,2)

S3 = √(-2-1)²+(2-(-1))²

S3 = √(-3)²+(3)²

S3 = √9+9

S3 = √18

S3 = 3√2

Side FG, F(-2,2), G(-4,0)

S4 = √(-4-(-2))²+(0-2)²

S4 = √(-2)²+(-2)²

S4 = √4+4

S4 = √8

S4 = 2√2

Side GA, G(-4,0), A(-2,3)

S5 = √(-2-(-4))²+(3-0)²

S5 = √2²+3²

S5 = √4+9

S5 = √13

Solve for Perimeter

P = S1 + S2 + S3 + S4 + S5

P = 2 + √13 + 3√2 + 2√2 + √13

P = 16.28

Answer16.28 units
