23. explain the difference of JPEG vs. GIF vs. PNG vs. BMP vs. TIFF


JPEG

(Joint Photographic Experts Group)

is a File Format with a lossy compression. It allows some data loss for a smaller file size. It compresses grayscale and color information but allows for images to be displayed in 16,000,000 colors.

GIF

(Graphics Interchange Format)

is a File Format with a lossless compression. Meaning, there is no data lost in the compression. GIF supports animation and is preferred for images with only a few colors. (GIF only allows for 256 colors)

PNG

(Portable Network Graphics)

is a File Format that takes off from the transparency capabilities of GIF, primarily developed due to GIF's licensing requirement.

TIFF

(Tagged Image File Format)

is a File Format that is a flexible bitmap image that is supported by many software. It has a much larger file size compared to JPEG and GIF and is commonly recommended by professional printers since it is supported by most press machines.

BMP

(Bitmap)

The standrad Windows file format on DOS and Windows–compatible platforms. It has a lossless compression.

24. It converts RAW images (uncompressed images usually from digital camera to JPEG.


CONVERTING RAW IMAGES TO JPEG FORMAT

ANSWER:

If you're wondering if you can convert RAW photographs (uncompressed images often taken with a digital camera) to JPEG, the answer is YES, and you can do it using Adobe Photoshop on your computer. RAW photos, on the other hand, are higher-quality, unfiltered photographs. JPEG images have a certain degree of processing baked in at the time of capture that can't be removed. When converting from raw to jpg, you lose the ability to manipulate the image further.

EXPLANATION:

To convert RAW to JPEG format, first, open the photographs in a RAW mode in a program like Photoshop. Choose 'Save As' from the 'File' menu, then '.jpg' from the drop-down list (it might appear as JPEG). Next, choose a compression ratio of 90-100 percent; otherwise, quality will suffer. The file size is also reduced through compression. Make that the settings are RGB.

30. There are lossless and lossy file formats. Which of the file formats are better if you wantan almost complete image information? Why?​


ANSWER:

If I want an almost complete image information, I would choose lossless file formats.

WHY?

Because lossless file formats are as the name suggests "lossless", it does not lose (at most) any graphic information. For example, if you create an image from Photoshop and convert it to a lossless file format, what you see during post-processing is what you will see as an output of it. However on lossy file formats, as the name suggests "lossy" the graphical information from those file formats can degrade or will lose it's original quality from what you've produced/seen.

EXAMPLES:

Examples of lossless image file formats:

.png.raw.bmg

Examples of lossy image file formats:

.jpeg/.jpg.gif
