METAPHOR:

a figure of speech in which a word or phrase is applied to an object or action to which it is not literally applicable.

APOSTROPHE:

a punctuation mark ( ’ ) used to indicate either possession (e.g., Harry's book ; boys' coats ) or the omission of letters or numbers (e.g., can't ; he's ; class of ’99 ).

Ano-ano ang mga kahirapan na nasapit ng mga Pilipino sa kamay ng mga kastila?

Isa sa mga dayuhang sumakop sa ating bansa ay ang mga Kastila. Nang dahil dito, naranasan ng ating mga ninuno ang kahirapan sapagkat ginawa nilang alipin ang mga Pilipino. Ang mga inosenteng Pilipino ay nadadawit sa mga kahasulang nagagawa ng mg mananakop kung kaya't sila ang napaparusahan, binubugbog hanggang sa mawalan ng hininga o patayin sa harap ng nakararami. Isa rin sa mga kahirapan na naranasan ng mga Pilipino sa kamay ng mga Kastila ay ang kawalan ng hustisya, kawalan ng kalayaan at ang diskriminasyon sa sariling lupain.
