14. ano ang mga ginawa ng mga babae bago ang kolonyalismo?​


Answer:

Ang mga babae ay nagkaroon ng malaking papel sa mga komunidad bago ang kolonyalismo. Ang mga babae ay naging mga tagapag-alaga ng mga bata, tagapag-alaga ng mga hayop, tagapagtanim, tagapagluto, tagapag-ayos ng mga kasanayan sa paggawa.


15. ano ang gampanin ng mga babae bago dumating ang mga kastila


Ang gampanin ng mga babae bago dumating ang mga kastila ay mataas ng katayuan ng babaeng Pilipino dahil sa pagkakapantay pantay ng karapatan na katulad sa mga lalaki. Ang kanilang ina ang nasusunod sa pagsasaad ng pangalan ng kanilang mga anak. Ang mga lalaki ang laging nasa likuran ng mga kababaihan kapag ang mga ito ang naglalakad na nagsisilbing proteksyon laban sa mga kaaway.
