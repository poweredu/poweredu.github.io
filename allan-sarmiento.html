10. allan and ben will share the ammount 960 pesos such that allan has twice as much as ben. how much is bens share?


Ben's share: x
Allan's share:  2x

x + 2x = 960

3x = 960

3x/3 = 960/3

x = 320

Substitute 320 to x:

Ben's share: x = Php 320
Allan;s share: 2x = 2(320) = Php 640

ANSWER:  Ben's share is Php 640.

Check:

320 + 640 = 960
960 = 960
