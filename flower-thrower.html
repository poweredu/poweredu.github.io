19. a ball is thrown vertically upward with an initial speed of 80 ft/s. how long will it take for the ball to return to the thrower?​


Answer:

The time for the ball to return to the thrower's hand is 5 seconds.

Explanation:

In this problem, we will be using the concept of free-falling bodies. The formula that we are going to use is:

Vf = Vi - gt

where

Vf     is the final velocity, the given unit is in ft/s

Vi     is the initial velocity, the given unit is also in ft/s

g      is the acceleration due to gravity, the unit to use is 32 ft/s²

        since the given initial speed is in ft/s

t       is the time traveled by the ball

Solving the problem

Let us solve for the time for the ball to go up by using the given formula above.

Vf = Vi - gt

Then substitute the given data

0 = 80 ft/s - (32ft/s²)t

Take note that the final velocity (vertical velocity) at maximum height reached by the ball is zero.

Transpose (32ft/s²)t to the left side of the equation

(32ft/s²)t = 80 ft/s

Divide both sides by 32ft/s²

t = (80 ft/s) / (32ft/s²)

t = 2.5 seconds

Now, to compute the time for the ball to return to the thrower's hand, we will multiply 2.5  seconds by 2 because the time going up is the same time going down.

Therefore,

t = 2.5 seconds x 2

t = 5 seconds

Therefore, the time for the ball to return to the thrower's hand is 5 seconds.
