1. Bethany has $3.10 in nickels and dimes. She has a total of 44 coins. What is the value of the nickels Bethany has?


Nickels = 5 cents

Dimes = 10 cents

Total number of coins = 44

Total value = $3.10 = 310 cents


Define x:

Let x be the number of nickels

The number of dimes = 44 - x


Solve x:

5x + 10(44-x) = 310

5x + 440 - 10x = 310

5x = 130

x = 26


Find the number of each type of coins:

Nickel = x = 26 coins

Dime = 44 - x = 44 - 26 = 18 coins


Find the value of the dime:

18 x 10 cents = 180 cents = $1.80


Answer: She has $1.80 worth of dimes
