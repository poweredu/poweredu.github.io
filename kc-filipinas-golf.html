Answer:

42

Step-by-step explanation:

To figure that out, we just list multiples of each number until we find one in common. Let’s start with 6:

6, 12, 18, 24, 30, 36, 42..And now, let’s do 7:

7, 14, 21, 28, 35, 42…And now, what’s the smallest number that these two lists have in common? Obviously, we can see it’s 42.
