11. maxine own 40% of the stock in millenium traders,Inc. Justine owns 15000 shares. Aleah owns all the shares not owned by maxine and justine . how many shares does maxine own if aleah has 25% more shares than maxine ?​


Answer:

Maxine owns 18,000 shares in Millennium Traders, Inc.

Step-by-step explanation:

Let's start by finding the total number of shares in Millennium Traders, Inc.:

If Justine owns 15,000 shares, and Maxine owns 40% of the stock, then the total number of shares is:

15,000 / 0.40 = 37,500

Aleah owns all the shares not owned by Maxine and Justine, which means she owns:

37,500 - 15,000 = 22,500

Now, we know that Aleah has 25% more shares than Maxine. Let's use "x" to represent the number of shares that Maxine owns:

x + 0.25x = 22,500

1.25x = 22,500

x = 18,000

Therefore, Maxine owns 18,000 shares in Millennium Traders, Inc.
