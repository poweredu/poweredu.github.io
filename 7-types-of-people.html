15. Two people went to the school canteen and ordered a burger. The menu has 7 types of burgers and both of them are equally likely to order any type. Find the probability that each person orders different type.​


Answer:

The number of ways for the first person to order a burger is n (7), and the number of ways for the second person to order a burger, given that it's a different type from the first person, is n-1 (6).

So, the probability that each person orders a different type is:

(n) * (n-1) / (n)^2 = (7) * (6) / (7)^2 = 6/49

So, the probability that each person orders a different type of burger is 6/49.

26. Tyler is catering a banquet for 250 people. each person will be served either a chicken dish that cost $5 or a beef dish that costs $7 each. Tyler spent $1500 total on on the banquet. how many dishes of each each type did Tyler' serve


Let's use C to represent the number of chicken dishes and B to represent the number of beef dishes served. We can set up two equations based on the given information:

C + B = 250 (Equation 1 - the total number of dishes served is 250)
5C + 7B = 1500 (Equation 2 - the total cost of the dishes is $1500)

We can use Equation 1 to solve for C in terms of B:

C = 250 - B

Substituting this into Equation 2, we get:

5(250 - B) + 7B = 1500
