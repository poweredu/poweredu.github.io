4. Amir can run 15 kilometers in one hour. How many kilometers can Amir run per minute? There are 60 minutes in 1 hour.


Answer:

0.25 kilometers

Step-by-step explanation:

Divide 15 by 60 to find the number of kilometers traveled by minute.

Answer:

From my calculations, Amir can run 0.25 KILOMETERS PER MINUTE.

It would be 0.25 KM because you would divide 15 by 60 ( 15 ÷ 60 ) to find how many he could run in a single minute. Your answer would be 0.25.

And now to check that work : you would multiply 0.25 with 60 ( 0.25 ÷ 60 ) and once you do that, you would get 15. So there you go. There's your answer.
