17. ano ang grid pattern?


Answer:

The grid plan, grid street plan, or gridiron plan (redirected from Grid pattern) is a type of city plan in which streets run at right angles to each other, forming a grid. The infrastructure cost for regular grid patterns is generally higher than for patterns with discontinuous streets.

29. Ano ang grid pattern?


Answer:

Ang isang grid ay isang network ng mga intersecting parallel na linya, totoo man o haka-haka. Karamihan sa mga kalsadang Amerikano ay inilalagay sa isang pattern ng grid, nangangahulugang ang mga kalye ay lumusot sa tamang mga anggulo at bumubuo ng isang pattern ng mga parisukat kung tiningnan mula sa itaas.

Answer:

Ang grid pattern sa globo o mapa ay mga pinagtagpu-tagpong mga guhit na nakatutulong upang matukoy ang lokasyon ng iba-ibang lugar sa globo.
