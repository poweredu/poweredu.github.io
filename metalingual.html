1) Metalingual:

This refers to the self-referential use of language. For example, in the sentence "'snow' has four letters," language is being used to talk about itself—that is, this is a metalingual function—as opposed to "snow is white" in which we are actually talking about snow.

2)When language is used to talk about language itself (code), the communication is metalinguistic. A good example would be: "What are you saying? ... By "metalinguistic function", we mean an utterance in which the addressee gives or ask for information about the code.

3. ano ang metalingual?


Answer:

Using a story to describe a term; similar to metalanguage, which is a language used to make claims about another language; metalingual — using one chunk of text to describe or clarify another. Hamlet's "To be or not to be" voice, for example, is the meta-language of despair.

4. what is metalingual function?​


Metalingual function of language is the ability of language to talk about its own features. Metalingual function of language becomes relevant in translation when a particular word is used in a special sense, deliberately a word play is done or linguistic ambiguity is created.
