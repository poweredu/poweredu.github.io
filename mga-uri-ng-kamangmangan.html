4. Paano nagkakaiba ang dalawabg uri ng kamangmangan?


Answer:

Dalawang Uri ng Kamangmangan

1. Kamangmangang madaraig vincible ignorance

2. Kamangmangang di madaraig invincible ignorance

Explanation:

Ang pagkakaiba nito ay ang madaraig (vincible ignorance) na kung saan gumagawa ng paraan ang isang tao upang malampasan ito. Ang pagkakaroon ng kaalaman dito ay magagawa sa pamamagitan ng pagsisikap o pag aaral. At ang di madaraig (invicible ignorance) na kung saan ang isang tao ay walang magawang pamamaraan upang malampasan ito. Ito ang uri ng kamangmangan na bumabawas o tumatanggal sa pananagutan ng tao sa kaniyang kilos o pasiya.

12. dalawang uri ng kamangmangan


Kamangmangan  

Answer:

Nadaraig o vincible ignorance

Ito ay isang uri ng kamangmangan na mayroon ang tao. Dito, gumagawa siya ng paraan upang malamapasan at mahigitan ito. Ang isang paraan ng paggawa ay ang pag aaral ng mabuti o pagbabasa ng aklat. Ang pagsisikap na makapatapos ng pag aaral ay isa ring halimbawa ng paraan para madaig ang uri ng kamangmangan na ito.

Di-nadaraig o invincible ignorance

Ito ang ikalawang uri ng kamangmangan. Dito, walang alam o pananagutan ang isang tao sa maaaring gawin ng iba. Wala siyang paraan upang malampasan ito. Ang isang halimbawa ay ang pagbibigay ng tulong sa mga tao sa lansangan. Paglipas ng ilang araw, ang mga nalimos ng tao ay gagamitin niya sa paggawa ng masama o hindi maganda

24. ano ang dalawang uri ng kamangmangan?


Answer:

Kamangmangan

Ang kamangmangan ay nagmula sa salitang mangmang na nangangahulugan na walang alam. Ang tao ay tinatawag na mangmang kung siya ay walang alam o walang ideya tungkol sa isang paksa na pinag uusapan. Gayunpaman, ang salitang ito ay hindi katanggap tanggap sapagkat ito ay isang paraan din ng pangiinsulto.

Ayon sa bibliya, ang kamangmangan o pagiging mangmang ay ang hindi wasto o maayos na paggamit ng paggamit ng tao sa kanyang isipan o kakayahang mag isip.

Ang dalawang uri ng kamangmangan:

Kamangmangan na madaraigKamangmangan na di madaraig

Kamangmangang madaraig

Kamangmangang madaraig o vincible ignorance ay ang kamangmangan kung saan ang isang tao ay gumawa ng pamaraan upang malampasan ito. Ang  pagkakaroon ng kaalaman dito ay nag uugat sa pagsisikap o pag-aaral.

Kamangmangan na di madaraig

Ang Kamangmangan na di madaraig o invincible ignorance naman ay ang kamangmangan kung saan ang tao ay walang magawang pamaraan upang malampasan ito. Ito ang uri ng kamangmangan na bumabawas o tumatanggal sa pananagutan ng tao sa kaniyang kilos o pasiya.

30. Ano ang uri ng kamangmangan ​


Nadaraig (vincible) - Ito ang kaalaman sa isang gawain subalit ang tao ay may pagkakataong itama o magkaroon ng tamang kaalaman kung gagawa ng pagkatataon na malaman at matuklasan.

Hindi nadaraig - kamangmangan dahil sa kawalan ng kaalaman na mayroon siyang hindi alam na dapat niyang malaman.
