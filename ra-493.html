4. which term is 493 given a1,=3&d=5​


Answer:

99th

HOW?

FORMULA to be used: an = a1+(n-1)d

Where: a1 = 3: d = 5: an = 493: n = ?

>> 493 = 3 + (n-1) (5)

MULTIPLY (N-1) (5) which is 5n - 5

>>> 493 = 3 + 5n -5

Add like terms, -5 by 3 which is -2

>>>493 = 5n -2

ISOLATE 5n by adding both sides of the equation by -2

493+2 = 5n (-2 + 2) <-- cancelled out

>>> 495 = 5n

ISOLATE n by dividing both sides of the equation by 5

495/5 = 5n/5 (5 gets cancelled out)

495/5 = n

495/5 is 99

The 99th term is 493 given by the rule of the sequence.
