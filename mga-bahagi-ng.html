2. isulat ang mga bahagi ng computer isulat ang mga bahagi ng computer ​


Answer:

1) Computer Processor

Kabilang sa kahulugan ng computer hardware ang parehong panloob at panlabas na mga aparato. Ang mga panloob na aparato ay madalas na tinutukoy bilang mga sangkap. Sila ay naninirahan sa motherboard, na siyang pangunahing circuit board ng computer. Ang sentral na yunit ng pagproseso, o CPU, ay ang tinatawag na utak ng kompyuter na nagpapahiwatig ng mga tagubilin sa code at kumokontrol sa lahat ng mga operasyon ng computer. Naglalabas din ito ng mga simpleng kalkulasyon at gumagalaw ang data sa loob at labas ng memorya.

2) Computer Secondary Storage

Ang pangalawang imbakan ay tumatagal ng lugar sa mga permanenteng memory device tulad ng mga hard drive at flash drive. Ang data na nakaimbak sa mga aparatong ito ay mananatili kahit na ang computer ay naka-off. Maraming mga hard drive ay tinatawag na panloob dahil sila ay matatagpuan sa loob ng kaso ng computer. Mayroon ding mga panlabas na hard drive na nakakonekta sa isang computer sa pamamagitan ng USB port.

3) Computer Hardware Input / Output Devices

Ang isang computer ay may input at output device para sa komunikasyon sa labas ng mundo. Hinahayaan ka ng device ng input na ipasok mo ang data sa computer. Ang isa sa mga pinaka-karaniwang hardware input device ay ang keyboard, na ginagamit para sa text input. Ang mouse at touch screen ay mga input device din; ang kanilang layunin ay upang payagan ang isang gumagamit ng computer na ilipat ang isang cursor sa isang screen. Ang mga camera, scanners at microphones ay maaari ding gamitin para sa input ng computer ng mga partikular na uri ng data. Ang mga monitor na display, speaker at 2D at 3D printer ay ilan sa mga pinaka karaniwang mga aparatong output ng computer.

4) Pangunahing Imbakan

ang CPU ng pangunahing imbakan, na tinutukoy din bilang pangunahing memorya o random na memory ng access (RAM), upang mag-imbak ng mga tagubilin ng programa at anumang data na maaaring mangailangan ng mga tagubilin. Ang impormasyon na nakaimbak sa RAM ay pansamantalang likas at mananatili lamang habang tumatakbo ang kasalukuyang programa. Kapag pinapatay mo ang computer, na-clear ang pangunahing memorya. Available ang permanenteng memorya sa pangalawang imbakan.

5) Ang mga flash drive ay mas mabilis kaysa sa mga hard drive at ginagamit para sa panloob na patuloy na imbakan sa mga smart phone at tablet. Tulad ng mga panlabas na hard drive, ang mga external flash drive ay maaaring konektado sa mga computer sa pamamagitan ng USB port. Ang ganitong uri ng flash drive ay maaari ding tinukoy bilang isang thumb drive o jump drive.

MGA BAHAGI NG EDITORYAL

1.Panimula-Ito ang paksa o balitang batayan ng isusulat na tudling.Narito ang mga tala o detalye ng paksa.Kailangang ito ay maging maikli lamang.

2.Katawan.Ito ang kuru-kuro o palagay ng sumulat ukol sa paksa.Maaaring laban o sang-ayon siya sa paksa.Narito rin ang layunin ng sumulat ngeditorial.

3.Panapos o pangwakas-Pagpapatibay ito ng kuru-kuro at pagbibigay ng mungkahi o solusyon sa tinatalakay na isyu.

24. Dalawa ang bahagi ng talinghaga: ang bahagi ng mga paanyaya, at ang bahagi ng piging. Ipakita kung saan nagsisimula at natatapos ang mga ito.​


Answer:

Sa misa ay sinasabi ito ng pari: "Ito ang Kordero ng Diyos … Mapapalad ang mga inaanyayahan sa kanyang piging". Ang piging ng Panginoon ay yaon ding piging ng kasalan ng Kordero at ng kanyang Iglesya na binabanggit sa Apokalipsis 19. Sa talinghaga ng Mateo 22:1-14, ito ay ipinahihiwatig ng piging na inihanda ng hari para sa kasal ng kanyang anak. Bagama’t binuksan din ng hari ang piging para sa lahat, "masama man o mabuti", hindi naman lahat ay maaaring manatili ditong walang tamang damit.
