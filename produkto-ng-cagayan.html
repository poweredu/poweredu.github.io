16. katangiang ng cagayan​


Answer:

Katangian ng Cagayan

Ang lalawigan ng Cagayan ay matatagpuan sa Rehiyon 2 o lambak ng Cagayan, kasama ang mga lalawigan ng Batanes, Isabela, Nueva Vizcaya, at Quirino. Ang kabisera ng lalawigang ito ay ang lungsod ng Tuguegarao.

Explanation:

Kung ilalarawan natin ang katangian ng Cagayan, narito ang ilan sa mga mahahalagang bagay na kailangan mong tandaan:

Malaking bahagdan ng lalawigan ang lambak, at ang mga kapatagan dito ay nabuo dahil sa mga sediment na dala ng Ilog Cagayan.Sa kanlurang bahagi ng lalawigan matatagpuan ang mga bundok ng Cordillera.Sa silangang bahagi naman ng lalawigan matatagpuan ang mga bundok ng Sierra Madre.Sa hilaga ng lalawigan makikita ang mga pulo ng Babuyan, na sakop pa rin ng Cagayan.May isang aktibong bulkan sa lalawigan – ang bulkang Caguas a bayan ng Gonzaga.
