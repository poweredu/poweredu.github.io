Ang mga Roman ay kinikilala bilang pinakadakilang mambabatas ng

sinaunang panahon. Ang kahalagahan ng Twelve Tables ay ang katotohanan na

wala itong tinatanging uri ng lipunan. Ito ay batas para sa lahat, patrician o

kaukulang parusa. Nakasaad dito ang mga karapatan ng mga mamamayan at

PAMUMUHAY

KONTRIBUSYON

Ang mga Roman ay nagpakita rin ng kanilang galing sa inhenyeriya.

Nagtayo sila ng mga daan at tulay upang pag-ugnayin ang buong imperyo

kabilang ang mga nasa malalayong lugar. Karamihan sa mga daang ginawa nila

GUSALI

KONTRIBUSYON

Ang mga Roman ang nakatuklas ng paggawa at paggamit ng semento.

pantakip sa labas ng pader. Umaangkat sila ng marmol mula sa Greece.

ginagamit sa mga temple, aqueduct, at iba pang mga gusali. Isa sam mga uri ng

gusaling ipinakilala ng mga Roman ay ang basilica, isang bulwagan ng

nagsisilbing korte at pinagpupulungan ng Assembly. Ang forum ay tumutukoy

Answer:

Kontribusyon ng Kabihasnang Romano

Pulitika

Sa mga Romano nagsimula ang ideya ng pagtatayo ng isang senado. Ang corvus at ang harpax ay mga taktikang ginagamit ng mga Romano sa pagigipaglaban. Ang testudo ay isang posisyon ng mga sundalong Romano na nagpoprotekta sa isang grupo ng mga mandirigma. Sila ang nagsimula ng pakikidigma gamit ang siege weaponry.

Pamumuhay

Ang mga Romano ang nagpakilala sa atin ng mga Roman Numerals na ginagamitan ng mga letrang I, V, X, L, C, D, at M. Ito ay ginagamit sa napakaraming bagay sa kasalukuyan, katulad nalang sa agham, akademya, at pamahalaan. Unang ginawa din ng mga Romano ang mga dyaryo. Ito ay ginagamit upang magbahagi ng impormasyon sa mga mamamayan. Sila ang unang gumamit ng kalendaryong Julian. Naimbento ng mga Romano ang isang sistema ng postal service.

Gusali

Ang mga Romano ang unang gumawa ng mga maaayos at modernong plumbing system. Dahil dito, may malinis na tubig na dumadaloy sa ating mga kabahayan. Sila rin ang lumikha ng mga aqueduct upang mas mabilis ang paglalakbay ng tubig mula sa mga sources papunta sa mga bayan at lungsod. Ang hypocaust system naman ang naging daan upang maimbento ang mga heater. Pinatibay din nila ang mga semento na tumatagal ng ilang libong taon. Nagtayo din ang mga Romano ng malawak na network ng mga kalsada.
