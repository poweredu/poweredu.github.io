Thank you for providing more information! Based on your message, it seems that you're asking for help with constructing a frequency distribution table using the data provided. Here's how you can do it:

1. Determine the range of the data by subtracting the smallest value from the largest value: Range = Largest Value - Smallest Value = 26 - 13 = 13

2. Determine the number of classes you want to use in your frequency distribution table. A good rule of thumb is to use between 5 and 20 classes, depending on the size of your data set.

3. Determine the width of each class interval by dividing the range by the number of classes: Class Width = Range / Number of Classes

4. Round up the class width to a convenient number.

5. Determine the lower limit of the first class interval by subtracting one class width from the smallest value.

6. Construct a table with two columns (Class Interval and Frequency).

7. Fill in each row of the table with the following values:

- Class Interval: The range of values that fall into this class interval

- Frequency: Count how many values in your data set fall into this interval
