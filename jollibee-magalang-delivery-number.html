24. A store received a delivery that consists of x boxes that weigh 30 pounds each and y boxes that weigh 20 pounds each. There are 64 total boxes in the delivery and they have a combined weight of 1400 pounds. Enter the number of boxes in the delivery that weigh 20 pounds.


Hi Jaidynbobo1,

Solving this type of problem requires a bit of critical thinking and some slice of analysis. We can also use methods in algebra which are substitution and elimination methods. But understanding the question very well gives us a glimpse on how this kind of problem be solved. First you need to establish what are the given from the problem. From there you can establish equations based on the statements given. And lastly, you can use the above said methods to identify the unknown variables.  

Here’s how it’s done;

First identify the variables and the equations from the problem stated;

X = boxes that weigh 30 pounds each

Y = boxes that weigh 20 pounds each

For the statement “There are 64 total boxes in the delivery…” we can generate the equation

[tex]x + y = 64[/tex]

For the statement “they have a combined weight of 1400 pounds” we can have another equation

[tex]30x+ 20y = 1400[/tex]

Now we have two unknowns and two equations. This problem can be solved.

Next, from the given set of equations we should rearrange the first one with the variable Y be left on one side of the equation. See the process below  

[tex]x+y=64[/tex]

Subtract X in both side of the equation we have

[tex]x+y-x=64-x[/tex]

The variable x is eliminated from the left side of the equation yielding to a new derived equation

[tex]y=64-x[/tex]

Next, impose this new derived equation to the second equation. Here's how it goes.

[tex]30x+ 20y = 1400[/tex]

[tex]30x+ 20(64-x) = 1400[/tex]

distributing x to 64 and -x

[tex]30x+ 1280-20x = 1400[/tex]

transposing 1280 to the right side of the equation

[tex]30x+ 1280-20x -1280= 1400-1280[/tex]

combining like terms

[tex]10x = 120[/tex]

dividing both side by 10

[tex]\frac{10x}{10} = \frac{120}{10}[/tex]

[tex]x = 12[/tex]

Having the value for x, we can now solve for the value of y. Using this answer to the first equation using substitution method.

[tex]y=64-x[/tex]

Substituting the value of x

[tex]y=64-(12)[/tex]

[tex]y=52[/tex]

Referring from the given variables that we have established on the first part of this solution, we can conclude that;

There are 52 boxes that weigh 20 pounds each and there are 12 boxes that weigh 30 pounds each.

Hence the answer to our question is 52

You can verify your answers by substituting the values for x and y to the second equation. If the values on both sides are equal then it is correct. Just make it sure that

[tex]x=12[/tex] and

[tex]y=52[/tex]
