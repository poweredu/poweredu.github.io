5. if angle C and angle D are supplemet angle which measures are 2x and 2x + 4 ,respectively. What is the measures of angle D?


Answer:

     Angle D = 92

Step-by-step explanation:

Given:  Angle C = 2x

            Angle D = 2x + 4

Determine the value of x.

Note that two angles are said to be supplementary, when these angles sums up to 180 degrees.

Therefore, angle C + angle D = 180 degrees.

Substitute the equations to the unknowns:

      ( 2x ) + (2x + 4 ) = 180

                     4x + 4 = 180

                           4x = 176

                              x = 44

Determine angle C and D.

Since the value of x is 44, angles C and D can now be already determine by simply substituting the value of x to the equation.

     Angle C = 2x = 2(44) = 88

     Angle D = 2x + 4 = 2(44) + 4 = 88 + 4 = 92

Therefore, Angle D = 92.


Check:

          ( 2x ) + (2x + 4 ) = 180

                       88 + 92 = 180
