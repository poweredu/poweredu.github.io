3. What are the roles of the project, program, and portfolio managers? what are suggested skills for project managers? what additional skills do program and portfolio managers need?


The roles of

Project manager : To overlook and to meet the project objectives with the resources available within the schedule and budget.Program manager : The program manager focuses on managing the projects through strategy and to make sure that the projects are moving in the right direction.Portfolio manager : The manager needs to make sure that the resources, budget, space, people, and equipment are enough for a project, and programs to be done right.

Skills for :

projects manager :  effective communication, negotiation, scheduling and time management, leadership, and technical expertise.Program manager : good knowledge of planning, monitoring and controlling programs, the ability to solve problems, and good knowledge of budgeting.Portfolio manager : humility, strong emotional control, decisiveness, anticipation, analytical ability and tenacity.

Further Explanation :

In an organization it is essential that the project, program, and portfolio managers understand the roles that each other plays in brining the successful completion of a strategic goal. It is to be effective, and to avoid unnecessary confusion and miscommunication that might occur.

Understanding the roles of project, program, and portfolio managers :

Project manager : A project refers to the process of creating a unique product, service or result, the project manager is a key element of a project, the project manager job is to balance the scope of work and to meet the project objectives with the resources available within the schedule and allotted budget. They must ensure the project meets the quality required by the customers or clients. The skills that a project manager must have are; effective communication, negotiation, scheduling and time management, leadership, and technical expertise.Program manager : Program refers to a group of projects that are managed in a coordinated way to ensure that value is achieved. The program manager is to manage this multiple projects, the program manager ensure that the projects are moving in the right direction and are working according to the plan and strategy. The skills that a program manager must have are; good knowledge of planning, monitoring and controlling programs, the ability to solve problems, and good knowledge of budgeting.Portfolio manager : Portfolio refers to the group of projects and programs that are managed as a group to achieve strategic objectives. The portfolio manager is the centralized management of one or more portfolios to achieve an organization's strategic objectives, the manager must organize the budget, people, resources, space and equipment in order for several projects and programs to be done. The skills a portfolio manager must have are; humility, strong emotional control, decisiveness, anticipation, analytical ability and tenacity.

7. Which are the best examples to include in a career portfolio? photograph, a current resume, and a birth certificate ethnic identity, professional recommendations, and a cover letter birth certificate, current work samples, and a photograph work samples, professional recommendations, and a resume


A career portfolio should have a good sample of the person's work so that any potential employer could make the right decision when thinking of hiring. In the Philippines, what is usually needed is a resume, sample works (of course the best ones), photographs of work samples, professional recommendations and an application cover letter. There is no need for an ethnic card since we do not have that anyway, unless part of the IP in the Philippines. The same goes for a birth certificate or an ID  photo. These could be asked for a part of the employee record but should not as part of the work portfolio.

When assembling a career portfolio make sure you include the best work you have ever done. If the work could have a short description, better. Also, make sure you have professional recommendations or testimonials from previous clients. Also make sure that your Facebook profile looks clean and professional. Do not use dummy accounts or accounts with inverted names (like NauJ Aled Zurc).
