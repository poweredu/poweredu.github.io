For example if we advertise on Inst4gram we get 100,000 reach and 150,000 impressions. From this data, it means that there are 100,000 people who see our advertisements more than once, so that the total number of advertisements that appear in the audience is 150,000 times.

The benefits of ‘using reach and impressions as parameters for digital campaigns are how many audiences the ad has reached and whether or not the strategy has reached the desired target. What does the page break preview show?Page Break Preview. Shows how the range of the pages to print will be. While in Page Break Preview, you can use the mouse to adjust page breaks, or insert/remove them.

29. Print preview allows a review of the document and the implementation of necessary changes before printing. True or False?


Answer:

True.

Explanation:

Print preview is a feature in some software applications that allows users to view their document in the format it will appear when printed. This feature allows users to review the document layout, font styles, images, and other elements before printing, which can help them identify any errors or formatting issues that need to be corrected. Additionally, print preview can also provide users with the option to make adjustments to the document, such as adjusting margins or page orientation, before sending it to the printer. Overall, print preview is useful for ensuring that the document is printed correctly and is ready for distribution or dissemination.
