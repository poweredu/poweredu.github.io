1. GIF supports 16 million colors


Answer:

The format supports up to 8 bits per pixel for each image, allowing a single image to reference its own palette of up to 256 different colors chosen from the 24-bit RGB color space. It also supports animations and allows a separate palette of up to 256 colors for each frame. These palette limitations make GIF less suitable for reproducing color photographs and other images with color gradients, but well-suited for simpler images such as graphics or logos with solid areas of color.

Pictures

Typically, they are digital or electronic photographs or images that you have stored on a local storage device. There are three common types of image file formats. You can identify them by their filename extensions.

a. JPEG and JPEG

This is the abbreviation for.jpeg, which stands for Joint Photographic Experts Group. As with all other picture file extensions, it denotes the data compression method that makes the file Internet-compatible and transportable. This picture file type supports 16.7 million colours, making it suited for working with photographic images in full bloom. Unfortunately, it does not enable transparency, making it challenging to integrate photographs of this file type with other materials or parts of your project. However, if you're searching for the highest-quality image to include in your document, this is the image file format for you. JPG does not perform well with text, line art, or simple graphics. JPG photos are compact in terms of file size.

b. GIF

This is the abbreviation for Graphics Interchange Format. This image file format is capable of portraying transparency. Therefore, it can be combined with other elements or materials in your work. Additionally, it may display rudimentary animation. This might not be useful on printed material. Nonetheless, providing documents electronically via email or putting copies on a website could be fantastic. Because it can only support up to 256 colours, it is best suited for logos and art decors with a limited number of solid colours. GIF is superior for logos, drawings, small text, black-and-white images, and low-resolution files.

c. PNG

The correct pronunciation is "ping. It is an abbreviation for Portable Network Graphics. It was designed with GIF's capabilities in mind. It was developed primarily to expedite the transfer of photographs on the Internet. It is also excellent with transparency, but unlike GIFs, it does not support the motion. However, it can display up to 16 million colours. Therefore the visual quality of this image file format has also been much enhanced. PNG permits the control of an image's level of transparency or opacity.

12. Which image format is ideal for images with few colors and sharp edges? a. Bitmap b. GIF c. PNG d. JPEG


answer: B. GIF

The Graphics Interchange Format is a bitmap image format that was developed by American computer scientist Steve Wilhite while working at the bulletin board service provider CompuServe on June 15, 1987. 

1. For anything with sharp edges, fine lines and few colors (such as screenshots or logos) use:

- GIF if you're doing a webpage that has to support very old browsers (IE 6 mainly), or if you want to do simple animations

- PNG otherwiseThere is no good reason to use BMP, unless a specific program does not accept other formats.


2. For photographs, use:Camera-specific raw formats if you have professional equipment and want to do a lot of post-processing:

- JPEG otherwise (for publishing photos on the web, you'll have to convert the raw formats to JPEG as well)
