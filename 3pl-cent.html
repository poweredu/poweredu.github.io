12 pcs of 10-cent coins

14 pcs of 25-cent coins

Step-by-step explanation:

Let:

A = # of 5 cents

B = # of 10 cents

C = # of 25 cents

Therefore:

0.05A + 0.10B + 0.25C = 4.90 where:

B = 3A

C = B+2 = (3A+2)= 6A so:

0.05A + 0.10(3A) + 0.25(3A+2) = 4.90

0.05A + 0.30A + (0.75A + 0.50) = 4.90

1.10A = 4.90-0.50

1.10A = 4.40

A = 4.40/1.10

A = 4

B = 3A = 3(4) = 12

C = B+2 = 12+2 = 14

Checking:

0.05A + 0.10B + 0.25C = 4.90

0.05(4) + 0.10(12) + 0.25(14) = 4.90

0.20 + 1.20 + 3.5 = 4.90

4.90 = 4.90

18. A box contains 5-cent coins. Given that the total value of the coins is $7.35 and there are thirty-one 20-cent coins, find the number of 5-cent coins in the box.


31 pcs of 20-cent coins = $6.20

$7.35 - $6.20 = $1.15

$1.15 / 0.05 = 23 pcs of 5-cent coinsFormula:

Total Coins - 20 Cent Coins = Total Amount of 5 Cent Coins / 5 Cent Coins = Amount of 5 Cent Coins in the Box.

Given:

Total Coins: $7.35
20 Cent Coins: 31 [20*31 = 620 ($6.20)]

Use the Formula:

$7.35 - $6.20 = $1.15 / $0.05 = 23

Answer:

The Box Contains: (23) 5 - Cent Coins.
