1st equation: d = 50 - c

2nd equation: 4d+2c = 174

Substitute the value of d to the second equation

4d+2c = 1744(50-c) + 2c = 174200-4c+2c = 174200-2c = 174200-174 = 2c26 = 2c26/2 = 2c/213 = c

d = 50 - cd = 50 - 13d = 37
