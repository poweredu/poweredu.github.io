4. A vanilla cupcake costs twice as much as mocha cupcake. A vanilla and a mocha cupcake cost Php 73.50. How much is 3 mocha and 5 vanilla cupcake​


Vanilla cupcake= 2x
Mocha cupcake = x
Vanilla & mocha cupcake = Php73.50

To get the price of each cupcake, this will be the equation;
2x+x= 73.50

Solution
2x+x= 73.50
3x= 73.50

You need to divide 73.50 by 3 to get the price of the mocha cupcake .
73.50/3
= php24.50

The price of the mocha cupcake is Php24.50 so the vanilla cupcake will be Php49 because the price of it is twice the price of the mocha.

The question is how much is 3 mocha and 5 vanilla, so this will be the equation;

(3*Php24.50)+(5*Php49)
(‘*’ represents multiply)

Solution
3*24.50= 73.50
5*49=245
Php73.50 + Php245
= Php318.50

Answer
Php 318.50

After the first 100 cupcakes you purchase up to 150 cupcakes, the bakery will lower the price 13.50 pesos per cupcake. After you purchase 150 cupcakes the price will decrease to 10.00 pesos per cupcake. Write a function that models this situation.​


In mathematics, function define as a relation that associates for each element of a set x (domain of the function), to a single element of a set y (range of the function). In every input of value to x, there is corresponding value for an output to y.

The Given Problem:a for number of item for first 100b for number of item for another 150c for number of item for for another 150

To find its function related to the amount can be paid for every set of item, function in terms of price or amount.

x= 15(a), f(a)= 15a, a ≤ 100x= 13.50(b), f(b)= 13.5b, b ≤ 150x= 10(c), f(c)= 10c

If the function being asked is the total amount for buying the item.

y= f(a) + f(b) + f(c)

y= 15a + 13.5b + 10c

we must Define the Following: A.G.O.N.S.A Method

What are Asked? (Ano ang Tinatanong?)

What are the Given? (Ano ang mga clue?)

What Operation should be use? (Kailangan ba magdagdag, magbawas, paramihin o hatiin?)
